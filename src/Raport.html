1. Abstarct:

Raportul de fata surprinde detaliile care au vizat progresul proiectului 'Cuisine Web Helper' propus in cadrul laboratorului de Tehnologii Web.
Sunt surprinse in prezentul raport detalii legate de obiectivele proiectului si etapele realizarii acestor obictive, cat si date legate de folosirea Github.

2. Introducere:

In ziua de azi, tehnologia are un rol foarte important in vietile noastre, facilitand in principal cautarea de informatii. Oamenii nu mai cumpara carti de bucate, ci prefera sa caute retete pe internet. 'Cuisine Web Helper' se adreseaza oricarei persoane, fie aceasta pasionata de gatit, fie novice. Aceasta aplicatie este adecvata persoanelor care urmeaza o dieta speciala sau care tin post intrucat aceasta va recomanda retete in functie de campurile alese de utilizator, cum ar fi restrictii alimentare, regim, timp alocat.

3. Obiective:

  • Sa se dezvolte o aplicatie Web care sa vina in ajutorul persoanelor avand un regim alimentar special -- e.g., restrictii alimentare datorate unei boli (diabet, alergii la anumite substante), unui stil de viata (vegetarianism) sau practica spirituala (post).
  • Sistemul va recomanda in mod "inteligent" retete pe care utilizatorul le poate prepara in concordanta cu regimul alimentar, conform abilitatilor, timpului alocat.
  • Retetele recomandate vor fi expuse si in formatul JSON.

5. Modul de lucru cu Github

Sistemul de management al codului-sursă ales este Github,iar pentru a facilita lucrul cu acesta, am folosit Github Desktop. A fost necesara descarcarea aplicatiei, autentificarea cu contul de git ,crearea unui repository pentru proiect si clonarea proiectului. Modificarile facute in directorul proiectului sunt vizibile si pot fi adaugate proiectului prin succesiunea de comenzi 'commit' si 'push origin' care pot fi realizate prin simpla apasare a unor butoane din aplicatia Github Desktop. Pentru a prelua local schimbarile facute de ceilalti contribuitori se aplica, la fel de simplu, comenzile 'fetch origin' si 'pull origin'. Commit-urile au fost adaugate ('merge') branch-ului master.
